# Track your activities with PyClocks
PyClocks is a simple, lightweight PyQt5 application to track your activities.

Please go to [PyClocks web page](https://milessic.github.io/pyClocks/)

# Environment setup
 - Instal python3.12
 - install requirements.txt
 That's all!
